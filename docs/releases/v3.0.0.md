# v3.0.0 — AI Semantic Search

**Released**: 2026-05-30 | **Git tag**: `v3.0.0` | **Tests**: 698

Privacy-first AI semantic search layer. Adds natural language query translation, on-device vision tagging, cloud AI result enrichment, voice search, clipboard search, and natural language file operations. All AI features are OFF by default.

## Privacy model

- **AI OFF by default** — opt-in via `AIConfig` for every feature
- **Path sanitization** — usernames replaced with `~` before any metadata leaves the device
- **FileMetadataSummary** — structured, minimal metadata sent to cloud (never file contents in v3.0)
- Graceful degradation when cloud providers are unavailable — on-device features continue working

## Code review fixes at release

- Fixed actor isolation violation in HTTPSearchService
- Fixed TCP fragment buffering for HTTP requests
- Fixed speech recognition task leak risk (strong reference cycle)
- Cache eviction (TTL + size cap) added for long-running daemon stability

## Known issues

- CoreML embedding model (`paraphrase-multilingual-MiniLM-L12-v2`, ~470MB) not yet bundled — deferred to v3.1
- Cross-language search quality varies significantly by language pair
- Voice search accuracy degrades in noisy environments
- Cloud AI features require API keys and network — no offline fallback for translation/summarization
