# v2.0.0 — GUI + Extended Indexing

**Released**: 2026-05-30 | **Git tag**: `v2.0.0` | **Tests**: 435

Full-product milestone — adds native macOS GUI (Liquid Glass, Apple Intelligence glow, global hotkey), external/network volume indexing, media metadata extraction, and service integrations.

## What was built (cumulative from v1.0.0)

| Version | Focus | Key deliverables |
|---------|-------|-----------------|
| v2.0.0 | GUI + volumes | NSPanel + Liquid Glass, Apple Intelligence glow, global hotkey (Ctrl+Cmd+K), external/network volume indexing, LSUIElement menu bar app, Quick Look, right-click context menu, hotkey conflict detection |
| v2.1.0 | Media metadata | Image/audio/video/PDF metadata extraction, MediaMetadataIndex actor, FilterPipeline with metadata key parsing |
| v2.2.0 | Service integration | HTTP search (NWListener), URL scheme (`deepfinder://`), AppIntents (Shortcuts), AppleScript support |

## Architecture additions

```
Global Hotkey -> SearchPanel (NSPanel) -> IPC -> same Daemon -> same index
```

- **GUI**: NSPanel with `.glassEffect(.regular)`, AngularGradient glow (teal/violet/coral/amber, ~1.8s rotation, 60fps)
- **Same daemon** serves both CLI and GUI via the same Unix socket IPC
- **External volumes**: USB/Thunderbolt indexed on mount, index removed on unmount; SMB/AFP/NFS network volumes also supported
- **Media module** (`Sources/Media/`): 7 files for metadata extraction and indexing
- **Services module** (`Sources/Services/`): 4 files for HTTP, URL scheme, Shortcuts, AppleScript
- **IndexPersistence schema v2** with `metadata_json` column

## Known issues

- PID file TOCTOU race condition (carried from v1.0.0)
- GUI first-launch onboarding flow is minimal — no guided setup wizard
- Accessibility permission denial does not provide clear fallback instructions to users
