# v2.2 Service Integration Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development or superpowers:executing-plans.

**Goal:** Add HTTP search service, URL Scheme, Apple Shortcuts, AppleScript integration to DeepFinder.

**Architecture:** New `Sources/Services/` module. HTTP service via Network.framework NWListener (zero deps). URL Scheme via Info.plist + AppDelegate. AppIntents for Shortcuts. NSScriptCommand for AppleScript. All connect to daemon via IPC.

**Tech Stack:** Swift 6.2, Network.framework, AppIntents, PDFKit, NSScriptCommand

---

### Task 1: HTTPSearchService (REQ-2.2-01)

**Files:**
- Create: `Sources/Services/HTTPSearchService.swift`
- Test: `Tests/ServicesTests/HTTPSearchServiceTests.swift`

Tests:
- Server starts on specified port and listens on localhost
- GET /health returns 200 OK JSON
- GET /search?q=test returns JSON results array
- GET /stats returns daemon stats JSON
- Returns 404 for unknown routes
- Server stops cleanly

Implementation:
- NWListener on 127.0.0.1 with configurable port (default 7654)
- Simple HTTP/1.1 request parser
- JSON responses with CORS headers
- Actor-isolated, starts/stops cleanly
- Routes: /search, /stats, /health

### Task 2: URLSchemeHandler (REQ-2.2-02)

**Files:**
- Create: `Sources/Services/URLSchemeHandler.swift`
- Test: `Tests/ServicesTests/URLSchemeHandlerTests.swift`

Tests:
- Parse deepfinder://search?q=test → SearchURL.search(query: "test")

Implementation:
