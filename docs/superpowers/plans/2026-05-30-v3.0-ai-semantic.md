# v3.0 AI Semantic Search — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development or superpowers:executing-plans.

**Goal:** Add AI-assisted search capabilities — NL search, result summaries, search suggestions, semantic grouping, local vision, image similarity, cross-language search, privacy controls.

**Architecture:** New `Sources/AI/` module. AIModelProvider protocol with DeepSeek and Qwen cloud providers. FileMetadataSummary privacy boundary. Local vision via Vision framework. All AI features OFF by default.

**Tech Stack:** Swift 6.2, Vision framework, Speech framework, PDFKit, URLSession SSE, Security/Keychain

**Status**: ✅ All 16 REQs complete. Implementation followed this plan through Phases 1-4, with a completion sweep (Phase 5-6) covering KeychainStore, vision persistence, speech auto-trigger, NL operations undo, and REPL AI commands.
