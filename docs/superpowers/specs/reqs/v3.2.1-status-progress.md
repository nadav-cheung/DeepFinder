# REQ: daemon status 索引进度百分比 + ETA

**版本**: 3.2.1
**优先级**: P1

## 用户故事
作为用户，我希望 `deepfinder daemon status` 显示索引进度百分比和预计剩余时间，以便了解首次扫描还需要多久完成。

## 验收标准

**Given** daemon 正在执行首次全量扫描  
**When** 用户执行 `deepfinder daemon status`  
**Then** 输出包含：
- Files indexed: N / M (XX%)
- 若 M 未知则显示 "estimating..."

**Given** 扫描已完成  
**When** 用户执行 `deepfinder daemon status`  
**Then** 不显示进度百分比，仅显示文件总数

**Given** 扫描正在进行中，持续超过 10s  
**When** 用户执行 `deepfinder daemon status`  
**Then** 输出包含预计剩余时间 (ETA)

## 实现
- `Sources/Daemon/IPCProtocol.swift`: DaemonStats 加 `estimatedTotalFiles`
- `Sources/Daemon/DaemonMain.swift`: 扫描前预计数 + 进度追踪
- `Sources/CLI/DaemonCommands.swift`: 格式化百分比 + ETA

## 测试
- `Tests/DaemonTests/`: DaemonStats 编解码
- `Tests/CLITests/`: status 输出格式
